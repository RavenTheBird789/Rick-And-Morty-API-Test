# Rick-And-Morty-API-Test
Python program using a REST API to pull information from a website about Rick and Morty characters and return the information to the user in a specific format

To install, simply type "python3 " in your terminals command line

To run the program, either type "python3 rick_and_mortyAPI.py" in your terminals command line or use the alias command to create a shorcut to run the program in your terminal such as "alias run="python3 rick_and_mortyAPI.py"" and then type "run" in your terminal during your session
